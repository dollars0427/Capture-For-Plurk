Capture for plurk
---
這是一個利用Python寫成的螢幕截圖工具，讓你可以快速將自己的螢幕攝圖分享至Plurk上。
由於本工具使用了Mac OSX自帶的截圖功能，因此暫時只能夠在Mac上運行。Windows版本將會在未來釋出。

注意：目前本工具仍在Beta狀態，可能會有些許的BUG。

系統需求：Mac OSX 1.7.x Lion或以上
